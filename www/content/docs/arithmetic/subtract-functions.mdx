---
title: Subtract Functions
description: Subtract time units from datetimes
---

All subtract functions have the same signature as their add counterparts. They're provided for readabilityâ€”you can also use negative values with add functions.

## Available Functions

| Function | Description |
|----------|-------------|
| `subYears(input, years)` | Subtract years |
| `subMonths(input, months)` | Subtract months |
| `subWeeks(input, weeks)` | Subtract weeks |
| `subDays(input, days)` | Subtract days |
| `subHours(input, hours)` | Subtract hours |
| `subMinutes(input, minutes)` | Subtract minutes |
| `subSeconds(input, seconds)` | Subtract seconds |
| `subMilliseconds(input, milliseconds)` | Subtract milliseconds |
| `subMicroseconds(input, microseconds)` | Subtract microseconds |
| `subNanoseconds(input, nanoseconds)` | Subtract nanoseconds |

## Signature

All functions follow this pattern:

```ts
function subX(
  input: Temporal.Instant | Temporal.ZonedDateTime,
  amount: number
): Temporal.ZonedDateTime
```

## Examples

```ts
import { subDays, subMonths, subYears } from '@gobrand/tiempo';

const now = Temporal.Now.zonedDateTimeISO();

// Get yesterday
const yesterday = subDays(now, 1);

// Get last month
const lastMonth = subMonths(now, 1);

// Get 5 years ago
const fiveYearsAgo = subYears(now, 5);
```

## Equivalent Add Expressions

You can achieve the same results using negative values with add functions:

```ts
import { addDays, subDays } from '@gobrand/tiempo';

// These are equivalent:
subDays(now, 5);
addDays(now, -5);
```
