---
title: PlainDate Comparisons
description: Compare PlainDate values without time or timezone
---

For comparing calendar dates without time or timezone considerations.

## isPlainDateBefore

Returns true if the first date is before the second date.

```ts
function isPlainDateBefore(
  date1: Temporal.PlainDate,
  date2: Temporal.PlainDate
): boolean
```

```ts
import { isPlainDateBefore } from '@gobrand/tiempo';

const jan20 = Temporal.PlainDate.from('2025-01-20');
const jan25 = Temporal.PlainDate.from('2025-01-25');

isPlainDateBefore(jan20, jan25); // true
isPlainDateBefore(jan25, jan20); // false
isPlainDateBefore(jan20, jan20); // false
```

## isPlainDateEqual

Returns true if both dates are equal.

```ts
function isPlainDateEqual(
  date1: Temporal.PlainDate,
  date2: Temporal.PlainDate
): boolean
```

```ts
import { isPlainDateEqual } from '@gobrand/tiempo';

const jan20a = Temporal.PlainDate.from('2025-01-20');
const jan20b = Temporal.PlainDate.from('2025-01-20');
const jan25 = Temporal.PlainDate.from('2025-01-25');

isPlainDateEqual(jan20a, jan20b); // true
isPlainDateEqual(jan20a, jan25);  // false
```
